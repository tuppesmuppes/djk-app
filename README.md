# DJK Teamzulosung – PWA (GitHub Pages)

## Einmalige Einrichtung auf GitHub

1. **GitHub-Konto erstellen** (falls noch nicht vorhanden)
   https://github.com → kostenlos registrieren

2. **Neues Repository anlegen**
   → Grüner "New"-Button → Name: `djk-app` → **Public** → "Create repository"

3. **Alle Dateien aus diesem Ordner hochladen**
   Repository → "Add file" → "Upload files"
   Folgende Dateien hochladen:
   - index.html
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
   → "Commit changes"

4. **GitHub Pages aktivieren**
   Repository → Settings → Pages → Branch: main → Save

   Die App ist dann erreichbar unter:
   **https://dein-benutzername.github.io/djk-app**

---

## App installieren (auf jedem Gerät)

### Android (Chrome)
Die URL öffnen → unten erscheint Banner "Zum Startbildschirm hinzufügen"
Oder: Menü (⋮) → "App installieren"

### Windows (Edge oder Chrome)
Die URL öffnen → Adressleiste rechts: Install-Icon (⊕) klicken
Oder: Menü → "App installieren"

### iPad / iPhone (Safari)
Die URL öffnen → Teilen-Button → "Zum Home-Bildschirm"

---

## App aktualisieren

Einfach die neue `index.html` auf GitHub hochladen (Upload files → vorhandene Datei ersetzen).
Alle installierten Apps laden beim nächsten Start automatisch die neue Version.
